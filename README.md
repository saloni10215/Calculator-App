# Calculator App

## About

A modern Calculator web application built using HTML, CSS, and JavaScript. The application performs basic arithmetic operations and provides a clean, user-friendly interface with keyboard support.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Equals (=) operation
- Clear (C) button
- Backspace (⌫) button
- Keyboard support
- Error handling for invalid expressions
- Modern Apple-inspired UI
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Use

1. Click the calculator buttons or use your keyboard.
2. Enter numbers and operators.
3. Press **=** or **Enter** to calculate.
4. Press **⌫** or **Backspace** to remove the last character.
5. Press **C** or **Escape** to clear the display.

## Keyboard Shortcuts

| Key | Action |
|------|---------|
| 0-9 | Enter numbers |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| . | Decimal point |
| Enter | Calculate result |
| Backspace | Delete last character |
| Escape | Clear display |
